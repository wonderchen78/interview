# California-House-Price-Prediction
This is a regression problem to predict california housing prices.

The dataset contains 20640 entries and 10 variables.
*	Longitude
*	Latitude
*	Housing Median Age
*	Total Rooms
*	Total Bedrooms
*	Population
*	Households
*	Median Income
*	Median House Value
*	Ocean Proximity
